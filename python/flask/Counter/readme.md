# Counter App

Simple Flask counter app using sessions.

## Features

- Count page visits
- Add +2 to counter
- Reset counter
- Add custom number

## Technologies

- Python
- Flask
- HTML
- CSS

## Run Project

```bash
python3 server.py
```

Open:

```text
http://127.0.0.1:5000
```

This project helped me understand Flask sessions, routes, redirects, and form handling.
