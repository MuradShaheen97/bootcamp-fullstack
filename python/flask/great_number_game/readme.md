# Great Number Game

Simple Flask guessing game using sessions.

The app chooses a random number between 1 and 100.  
The user guesses, and the app tells them if the guess is too high, too low, or correct.

## Features

- Random number generation
- Flask sessions
- Form submission
- Too high / too low messages
- Play again button
- Attempts counter

## Built With

- Python
- Flask
- HTML
- CSS
- Jinja

## Run

```bash
python3 server.py
```
