# Checkerboard Flask Project

Simple Flask app that creates a checkerboard using HTML, CSS, and Jinja loops.
The app supports changing board size and colors from the URL.

Routes:

- `/` → 8x8 board
- `/4` → 4x8 board
- `/10/10` → 10x10 board
- `/10/10/green/yellow` → custom colors

Tech used:

- Python
- Flask
- HTML
- CSS
- Jinja
