# User Class Assignment

This assignment practices Python OOP by creating a `User` class.

The class has:
- `__init__` to create a user with a name and balance
make_deposit()` to add money
make_withdrawal()` to remove money
display_user_balance()` to print the user balance
transfer_money()` to move money from one user to another

I created 3 users and tested deposits, withdrawals, balance display, and money transfer.
