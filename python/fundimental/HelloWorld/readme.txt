 Hello World – Python

 Overview

Basic Python assignment to practice:

 `print()`
 variables
string concatenation
 Key Points

 Print text:

```python
print("Hello World")
```

 Using variable:

```python
name = "Murad"
print("Hello " + name)
```

 Numbers need conversion:

```python
num = 42
print("Hello " + str(num))
```

Best way (f-string ):

```python
print(f"Hello {name}")
```
Run

```bash
python Hello-World.py
```

Learned

 Difference between string & number
 How to combine values
 Best practice: f-strings
