# Daikichi Routes

## Description

A simple Spring Boot application that demonstrates static and dynamic routes using `@RequestMapping` and `@PathVariable`.

## Technologies

- Java
- Spring Boot
- Maven

## Routes

| URL | Description |
|------|-------------|
| /daikichi | Welcome message |
| /daikichi/today | Today's fortune |
| /daikichi/tomorrow | Tomorrow's fortune |
| /daikichi/travel/{city} | Travel message using a path variable |
| /daikichi/lotto/{number} | Fortune based on whether the number is odd or even |

## Features

- Static routes
- Dynamic routes with `@PathVariable`
- Conditional logic using `if` statement
- URL path variables

## How to Run

1. Run the Spring Boot application.
2. Open your browser.
3. Visit any of the routes above.
